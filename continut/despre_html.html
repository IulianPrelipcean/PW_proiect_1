Limbajul HTML

HTML(HyperText Markup Language) este un limbaj de marcare utilizat pentru crearea paginii web.

Un exemplu de cod html il gasiti in imaginea urmatoarea care se afla la adresa http://upload.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png.
Imaginea are atributul height de valoarea 147.

Tag-urile html cele mai utilizate in realizarea tabelelor sunt:

table defineste un tabel
tr un rand intr-un tabel
td o celula intr-un tabel

Observatie: Pentru a pune un contur unui tabel se poate(dar nu se recomanda) sa se puna abtributul border="1".

La validare va da eroarea: The border attribute on the table element is obsolete.

Cerinta 3

Pentru a crea liste se pot folosi tag-urile:

  1. ul - lista neordonata
  2. ol - lista ordonata
  3. li - item din lista
  4. dl - lista cu descrieri
  5. dt - termen dintr-o lista cu descrieri
  6. dd - descrierea dintr-o lista cu descrieri

Pentru a insera un document HTML in altul se poate folosi tag-ul <iframe>.
Astfel, exemplu de mai jos inserează conţinutul de la adresa https://html.spec.whatwg.org/multipage/.
